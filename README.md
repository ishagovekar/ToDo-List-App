# 📝 To-Do List App

A simple and user-friendly To-Do List desktop application built using Python and Tkinter.

This application helps users manage daily tasks efficiently with features like adding, deleting, and marking tasks as completed.

---

# 🚀 Features

✅ Add new tasks  
✅ Delete tasks  
✅ Mark tasks as completed  
✅ Simple and clean GUI  
✅ Beginner-friendly Python project  

---

# 🛠 Technologies Used

- Python
- Tkinter
